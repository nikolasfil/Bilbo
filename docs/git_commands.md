```bash
git checkout -b <new_branch_name > #creates a new branch and switches to it

git branch -d <branch_name> #deletes the local branch 

git push -d <branch_name> #deletes the remote branch

git merge <branch_name>  #it will merge the branch you called with the one you are currently at 

git switch <branch_name> #switches to the branch you called, if it doesn't exist locallt it will download it
```
